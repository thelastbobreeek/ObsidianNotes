Конструктор - это метод, вызываемый при создании экземпляра класс или структуры. Конструктор может иметь [[Перегрузка|перегрузки]] - несколько вариаций в зависимости от аргументов. Конструкторы позволяют обеспечить корректность экземпляров типа, когда они создаются.
## Синтаксис конструктора
Конструктор - метод с тем же именем, что и его класс. Его сигнатура может включать необязательный [[Модификаторы доступа|модификатор доступа]], имя метода и список параметров. Он не включает тип возвращаемого значения. Вот пример конструктора для класса `Person`
```cs
public class Person
{
   private string last;
   private string first;

   public Person(string lastName, string firstName)
   {
      last = lastName;
      first = firstName;
   }

   // Remaining implementation of Person class.
}
```
### Наследование в конструкторе
Для вызова конструктора родителя стоит использовать ключевое слово `base`.
