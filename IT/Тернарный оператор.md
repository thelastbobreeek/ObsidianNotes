Условный оператор ? : в [[CSharp|C#]], также называемый тернарным, вычисляет логическое выражение и в зависимости от полученного результата true или false возвращает результат одного из двух соответствующих выражений, как показано в примере:
``` cs
string tempInCelsius < 20.0 ? "Cold." : "Perfect!";
```

Синтаксис тернарного оператора выглядит следующим образом:
```cs
condition ? consequent : alternative
```
