`Stack<T>` - **коллекция** в [[CSharp|C#]], хранящая значения в формате **LIFO**(последний положил - первый забрал). Как **стопка книг** на столе - не **разобрав** **верхние**, не **достать** **нижние**.
## Основные свойства
1. Принцип **LIFO**(Last In - First Out)
   ```cs
   var stack = new Stack<int>();
stack.Push(1);
stack.Push(2);
stack.Push(3);
stack.Pop(); // 3
   ```
2. **Нет** доступа по **индексу**. Работа идет только с **вершиной** стека.
   ```cs
   stack[0]; // ❌ нельзя
   ```
3. **Быстрые** операции - 0(1)
   ```cs
   stack.Push(item); // добавить
stack.Pop();      // снять
stack.Peek();     // посмотреть верхний
   ```
4. Строгий **порядок** - элементы выходят в **обратном** **порядке** добавления
   ```cs
   push: 1 → 2 → 3
pop:  3 → 2 → 1
   ```
## Методы
- Push(T item) - **добавить** элемент на **вершину**
- Pop() - **получить** элемент из вершины и **удалить**
- Peek() - **получить** элемент из вершины **без удаления**
- .Count, Clear(), Contains(T item) и т.д.
## Когда использовать
Когда **нужно**:
- **отмена** **действий**(Undo/Redo)
- **возврат** к предыдущему **состоянию**
- **рекурсия**/**DFS**
- временное **хранение** **состояний**
**Не** **подойдет** если:
- нужен **доступ** к **первому**(нижнему)
- нужен **поиск** по **элементам**
- нужен **порядок** добавления
