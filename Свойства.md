Свойство в [[CSharp|C#]] - это своего рода интеллектуальное поле в классе. 

Элемент, предоставляющий гибкий механизм для чтения, записи или вычисления значения поля данных, своего рода "посредник" между полями класса и внешним кодом, позволяющий контролировать доступ к данным объекта. 

Они выглядят как поля снаружи, но они реализуются как специальные методы. Это позволяет вызывающим классам легко получать доступ к данным и по-прежнему способствовать обеспечению безопасности и гибкости данных. Можно проверять правильность, делать отложенное вычисление и т.д.
## Синтаксис свойства
```cs
class Person
{
    private string _name; // приватное поле

    // Свойство для доступа к _name
    public string Name
    {
        get { return _name; }    // чтение
        set { _name = value; }   // запись
    }
}
```
- get вызывается при чтении свойства(person.Name)
- set вызывается при присвоении значения(person.Name = "Tim"), value - значение которое присваивают
## Автоматические свойства
```cs
public class Person
{
    public string Name { get; set; } // автоматически создается скрытое поле
}
```
При объявлении автоматического свойства [[Компилятор|компилятор]] создает скрытое поле резервного копирования для свойства. 
Выглядит это примерно так
```cs
private string _name;

public string Name
{
    get { return _name; }
    set { _name = value; }
}
```
Снаружи выглядит как поле, но внутри - методы. Можно добавить логику, не меняя внешний вид других классов.
