Queue - **коллекция** в [[CSharp|C#]], хранящая значения в формате очереди(**первый** вошел - **первый** вышел), как **очередь** в **поликлинике**. 
## Основные свойства
1. Принцип **FIFO**(First In - First Out) 
   ```cs
   var queue = new Queue<int>();
queue.Enqueue(1);
queue.Enqueue(2);
queue.Enqueue(3);
queue.Dequeue(); // 1
   ```
2. **Нет** **доступа** по индексу. Можно работать только с **началом** и **концом**.
   ```cs
  queue[0]; // ❌ нельзя
   ```
3. Операции **быстрые** - 0(1)
   ```cs
   queue.Enqueue(item); // добавить в конец
	queue.Dequeue();     // забрать из начала
	queue.Peek();        // посмотреть первый
   ```
4. Порядок **сохраняется** - элементы выходят **строго** в порядке добавления
   ```cs
   [a, b, c] → a → b → c
   ```
## Полезные методы
- `queue.Enqueue(item)` - **добавить** элемент в **конец** очереди
- `var item = queue.Dequeue()` - **получить** и **удалить** **первый** элемент
- `var item = queue.Peek()` - **получить** первый элемент **без** **удаления**
