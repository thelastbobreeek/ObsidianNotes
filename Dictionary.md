Dictionary - **коллекция** в [[CSharp|C#]], хранящая значения в виде "**ключ** -> **значение**". Он хранит **пары** значений. Каждому ключу соответствует **одно** значение, и **наоборот**.
## Основные свойства
1. `Dictionary` - **ассоциальный**(от **ассоциация**, не социальность) массив, он работает как "**ключ** -> **значение**", а не "**индекс** -> **значение**", как [[List]]. 
   ```cs
   var dict = new Dictionary<int, string>();
	dict.Add(1, "меч");
	dict.Add(2, "щит");
   ```
   - ключ(TKey) - **уникален**
   - значение(TValue) - **любое**
1. Ключи **уникальны** - одинаковых быть **не может**.
   ```cs
   dict.Add(1, "меч");
	dict.Add(1, "лук"); // ❌ исключение
   ```
   Если нужно **перезаписать** первый элемент:
   ```cs
   dict[1] = "лук"; // ок
   ```
2. Быстрый доступ по ключу - O(1), главное преимущество Dictionary.
   ```cs
   string item = dict[1];          // O(1)
	bool exists = dict.ContainsKey(1); // O(1)
   ```
3. Порядок **не гарантирован**. На текущих .Net часто совпадает с добавлением, но полагаться на это **нельзя**.
### Основные методы
```cs
dict.Add(key, value);     // если ключа нет
dict[key] = value;        // добавить или заменить

dict.Remove(key);
dict.Clear();
foreach (var pair in dict)
{
    var key = pair.Key;
    var value = pair.Value;
}
// или отдельно
dict.Keys;
dict.Values;
```
### Когда использовать
**Нужно**:
- **быстро** получать значение по **ключу**
- связать **объект** -> **данные**
- **кэшировать** что угодно

**Не** **подходит** если:
- нужен **порядок**
- нужен доступ по **индексу**, а не по **ключу**
- ключи **не уникальны**
