HashSet в [[CSharp|C#]] - коллекция **уникальных** элементов, основанная на **хэш-таблице**. 
Проще - это как `List<T>` но **без дубликатов** и с **быстрой** проверкой наличия объекта в списке.
## Основные свойства
1. **Уникальность** - в `HashSet<T>` не может быть одинаковых элементов. Например, 
   ```cs
   var set = new HashSet<int>();
   set.Add(1);
   set.Add(1); // не добавится
   ```
2. **Быстрота работы** - задачи **не зависят** от размера коллекции, в отличии от List и остальных, обозначается это как O(**1**(допустим, **секунды**)), в то время как у List обозначается как O(**n**), где n - кол-во элементов в списке. 
3. **Отсутствия порядка** в добавлении - элементы не хранятся в порядке добавления.
   ```cs
   var set = new HashSet<int> { 5, 1, 10 };

   // порядок перебора непредсказуем
   foreach (var value in set)
   {
       Console.WriteLine(value);
   }
   ```
4. Операции над множествами
   ```cs
	set.UnionWith(other);
	set.IntersectWith(other);
	set.ExceptWith(other);
	set.SymmetricExceptWith(other);
   ```
### Использование
`HashSet<T>` лучше использовать, когда:
- нужны **частые** проверки на наличие элемента
- нужны **уникальные** значения
- нужны операции над **множествами**
И лучше не использовать, когда:
- нужен **порядок**
- нужны **индексы** элементов
- допускаются дубликаты
