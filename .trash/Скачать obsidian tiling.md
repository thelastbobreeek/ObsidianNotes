# ✅ Вариант 1: Плагин **Pane Relief**

Позволяет делать то, что тебе нужно — **изменять размеры сплитов с клавиатуры**, как в тайлинговых менеджерах.

### Как поставить:

1. Settings → Community plugins → Turn on “Third-party plugins”
    
2. Browse → ищешь **Pane Relief**
    
3. Install → Enable
    

### Что он даёт:

- Увеличивать/уменьшать размеры панелей по горизонтали и вертикали.
    
- Навигацию между панелями.
    
- Закрывать/открывать.
    
- Настройку хоткеев для resize как в WM.
    

### Какие хоткеи настроить:

Settings → Hotkeys → ищи:

- **Pane Relief: Resize current pane left**
    
- **Pane Relief: Resize current pane right**
    
- **Pane Relief: Resize current pane up**
    
- **Pane Relief: Resize current pane down**
    

Поставь, например:

`Alt + H — уменьшить вправо Alt + L — уменьшить влево Alt + J — уменьшить вверх Alt + K — уменьшить вниз`

А-ля vim + hyprland.

# ✅ Вариант 2: Плагин **Obsidian Tiling**

Это экспериментальный плагин, который превращает Obsidian в что-то вроде i3/Hypaland внутри самого приложения.

### Возможности:

- Деление окон как в i3 (`vsplit`, `hsplit`).
    
- Resize через хоткеи.
    
- Навигация между окнами, как в vim/tmux.
    

### Минусы:

- Менее стабильный.
    
- Чуть хуже UX по сравнению с Pane Relief.
