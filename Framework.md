Framework - готовая **основа**, выступающая как каркас **действий** и **архитектуры** в той или иной деятельности. Например, к **framework'ам** относятся: PARA - **каркас** идеи **распределения** **заметок** по **папкам**, [[Unity]] - **готовая архитектура** для создания **игр** - есть сцены, объекты, скрипты - разработчику не приходится писать это все **с нуля**.
