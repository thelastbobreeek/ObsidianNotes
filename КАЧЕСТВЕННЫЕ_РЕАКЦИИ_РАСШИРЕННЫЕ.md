# КАЧЕСТВЕННЫЕ РЕАКЦИИ: молекулярные, полные ионные и сокращённые ионные уравнения

Ниже для каждого сокращённого (чистого) ионного уравнения приведено:
1. **Молекулярное уравнение** — с типичными растворимыми солями/кислотами (в скобках указано агрегатное состояние).
2. **Полное ионное уравнение** — все растворённые ионы выписаны отдельно.
3. **Сокращённое (чистое) ионное уравнение** — только частицы, участвующие в образовании осадка/газа/основного продукта.

---

## 1) Сульфат: `Ba²⁺ + SO4²⁻ → BaSO4 ↓`

**Молекулярное (типичный вариант):**  
BaCl₂(aq) + Na₂SO₄(aq) → BaSO₄(s) ↓ + 2 NaCl(aq)

**Полное ионное:**  
Ba²⁺(aq) + 2 Cl⁻(aq) + 2 Na⁺(aq) + SO₄²⁻(aq) → BaSO₄(s) ↓ + 2 Na⁺(aq) + 2 Cl⁻(aq)

**Сокращённое (чистое) ионное:**  
Ba²⁺(aq) + SO₄²⁻(aq) → BaSO₄(s) ↓

---

## 2) Фосфат: `3Ag⁺ + PO4³⁻ → Ag3PO4 ↓`

**Молекулярное (типичный вариант):**  
3 AgNO₃(aq) + Na₃PO₄(aq) → Ag₃PO₄(s) ↓ + 3 NaNO₃(aq)

**Полное ионное:**  
3 Ag⁺(aq) + 3 NO₃⁻(aq) + 3 Na⁺(aq) + PO₄³⁻(aq) → Ag₃PO₄(s) ↓ + 3 Na⁺(aq) + 3 NO₃⁻(aq)

**Сокращённое (чистое) ионное:**  
3 Ag⁺(aq) + PO₄³⁻(aq) → Ag₃PO₄(s) ↓

---

## 3) Сульфид: `Pb²⁺ + S²⁻ → PbS ↓`

**Молекулярное (типичный вариант):**  
Pb(NO₃)₂(aq) + Na₂S(aq) → PbS(s) ↓ + 2 NaNO₃(aq)

**Полное ионное:**  
Pb²⁺(aq) + 2 NO₃⁻(aq) + 2 Na⁺(aq) + S²⁻(aq) → PbS(s) ↓ + 2 Na⁺(aq) + 2 NO₃⁻(aq)

**Сокращённое (чистое) ионное:**  
Pb²⁺(aq) + S²⁻(aq) → PbS(s) ↓

---

## 4) Образование сероводорода: `2H⁺ + S²⁻ → H2S ↑`

**Молекулярное (типичный вариант):**  
Na₂S(aq) + 2 HCl(aq) → H₂S(g) ↑ + 2 NaCl(aq)

**Полное ионное:**  
2 Na⁺(aq) + S²⁻(aq) + 2 H⁺(aq) + 2 Cl⁻(aq) → H₂S(g) ↑ + 2 Na⁺(aq) + 2 Cl⁻(aq)

**Сокращённое (чистое) ионное:**  
2 H⁺(aq) + S²⁻(aq) → H₂S(g) ↑

---

## 5) Карбонат — разложение кислотой: `2H⁺ + CO3²⁻ → H2O + CO2 ↑`

**Молекулярное (типичный вариант):**  
Na₂CO₃(aq) + 2 HCl(aq) → 2 NaCl(aq) + H₂O(l) + CO₂(g) ↑

**Полное ионное:**  
2 Na⁺(aq) + CO₃²⁻(aq) + 2 H⁺(aq) + 2 Cl⁻(aq) → 2 Na⁺(aq) + 2 Cl⁻(aq) + H₂O(l) + CO₂(g) ↑

**Сокращённое (чистое) ионное:**  
2 H⁺(aq) + CO₃²⁻(aq) → H₂O(l) + CO₂(g) ↑

---

## 6) Карбонат — выпадение карбоната кальция: `Ca²⁺ + CO3²⁻ → CaCO3 ↓`

**Молекулярное (типичный вариант):**  
CaCl₂(aq) + Na₂CO₃(aq) → CaCO₃(s) ↓ + 2 NaCl(aq)

**Полное ионное:**  
Ca²⁺(aq) + 2 Cl⁻(aq) + 2 Na⁺(aq) + CO₃²⁻(aq) → CaCO₃(s) ↓ + 2 Na⁺(aq) + 2 Cl⁻(aq)

**Сокращённое (чистое) ионное:**  
Ca²⁺(aq) + CO₃²⁻(aq) → CaCO₃(s) ↓

---

## 7) Галогениды (хлорид): `Ag⁺ + Cl⁻ → AgCl ↓`

**Молекулярное (типичный вариант):**  
AgNO₃(aq) + NaCl(aq) → AgCl(s) ↓ + NaNO₃(aq)

**Полное ионное:**  
Ag⁺(aq) + NO₃⁻(aq) + Na⁺(aq) + Cl⁻(aq) → AgCl(s) ↓ + Na⁺(aq) + NO₃⁻(aq)

**Сокращённое (чистое) ионное:**  
Ag⁺(aq) + Cl⁻(aq) → AgCl(s) ↓

---

Ag⁺(aq) + NO₃⁻(aq) + Na⁺(aq) + Br⁻(aq) → AgBr(s) ↓ + Na⁺(aq) + NO₃⁻(aq)

FeCl₃(aq) + 3 NaOH(aq) → Fe(OH)₃(s) ↓ + 3 NaCl(aq)

**Полное ионное:**  
Fe³⁺(aq) + 3 Cl⁻(aq) + 3 Na⁺(aq) + 3 OH⁻(aq) → Fe(OH)₃(s) ↓ + 3 Na⁺(aq) + 3 Cl⁻(aq)

## 13) Алюминий гидроксид (амфотер): `Al³⁺ + 3OH⁻ → Al(OH)3 ↓`

**Молекулярное (типичный вариант):**  
AlCl₃(aq) + 3 NaOH(aq) → Al(OH)₃(s) ↓ + 3 NaCl(aq)

**Полное ионное:**  
Al³⁺(aq) + 3 Cl⁻(aq) + 3 Na⁺(aq) + 3 OH⁻(aq) → Al(OH)₃(s) ↓ + 3 Na⁺(aq) + 3 Cl⁻(aq)

**Сокращённое (чистое) ионное:**  
Al³⁺(aq) + 3 OH⁻(aq) → Al(OH)₃(s) ↓

> Al(OH)₃(s) + OH⁻(aq) → [Al(OH)₄]⁻(aq)
