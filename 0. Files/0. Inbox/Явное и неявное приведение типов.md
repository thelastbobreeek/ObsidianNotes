Так как C# - статически типизированный язык, объявленные переменные нельзя переобъявить или записать в них значение другого типа. А для перевода одного типа в другой существуют явные и неявные преобразования типов. 

## Неявное
Неявное приведение осуществляется самим компилятором, без необходимого синтаксиса так как значения в переменной не теряются при преобразовании. 
Например, из int(32х битного числа) в double(64х битного числа), которое может вместить в себя все значения типа int. 
``` cs
int a = 3
double b = a
```
## Явное 
 Явное преобразование требует специального синтаксиса, так как при приведении типов могут потеряться значения переменной. 
 Например, при приведении типа float в int необходимо указать специальное слово, показывающее что надо привести переменную типа float в int несмотря на потерянную дробную часть.
 ``` cs 
 float a = 123.456
 int b = int(a) // 123
 ```
