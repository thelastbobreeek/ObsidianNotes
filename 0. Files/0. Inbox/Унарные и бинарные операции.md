Для преобразования переменных и чисел в [[C#]] существуют операторы. Но среди них есть две группы: унарные и бинарные.
## Унарные операции 
Унарный оператор работает только с одним операндом(что увеличивать/уменьшать). 
Берем одну переменную, и что то делаем с ней.
Например, 
```cs
int a = 5;
a++;
++a;
b--;
--b;
!true;
```
Они используют только один операнд(+ или - или !). Пусть даже и пишутся эти операнды два раза.
## Бинарные операции
Бинарные операторы же работают с двумя операндами. Это сложение, умножение, деление и т.д. Мы берем две переменные(операнда) и выполняем среди них операции. 
``` cs
a / 2;
b * a;
a - 1;
```
