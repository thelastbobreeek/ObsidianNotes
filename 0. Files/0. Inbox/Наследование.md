Наследование - возможность создавать классы **на основе** других классов, **сохраняя** свойства и методы класса-**предка**, но и с **возможностью** создать свои. 

Грубо говоря, это когда один класс является **"расширенной версией"** другого класса. "Умное копирование классов"

Помогает не писать **один и тот же** код по несколько раз, **расширить функционал** и **читабельность** кода.

Для **переопределения** реализации **логики** в дочернем классе, используется [[Модификаторы доступа|модификатор доступа]] **override**.
``` cs
public abstract class Animal
{
	public abstract void Speak();
}

public class Dog : Animal
{
	public override void Speak() => Console.WriteLine("Гав");
}

public class Cat : Animal
{
	public override void Speak() => Console.WriteLine("Мяу");
}
```
