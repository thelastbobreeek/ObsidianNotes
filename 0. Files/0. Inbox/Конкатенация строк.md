Конкатенация строк - процесс добавления одной строки в конец другой. Сцепление происходит с помощью оператора +. Для строковых литералов и строковых констант объединение происходит на моменте компиляции. Для строковых переменных же объединение происходит во время выполнения.

## Операторы + и +=
Для конкатенации строк используются операторы + и += для одиночного объединения. Даже если в одной инструкции используется несколько операторов +, содержимое строки копируется только один раз. 
### + при конкатенации строк
```cs
string a = "Hello";
string b = a + " world";
```
Что происходит: 
- создается новая строка
- копируется содержимое a
- добавляется " world"
### += при конкатенации строк
``` cs 
string a = "Hello";
a += " world";
```
+= делает то же самое, что и + с присваиванием:
``` cs
a = a + "world"
```
То есть: 
- старая строка "Hello" остается в памяти
- создается новый объект типа string
- ссылка a начинает указывать на него
