Что если нам надо сделать систему, где есть стреляющий и пуля. Стреляющий создает пулю и дает ей направление движения, куда надо двигаться. Но! Также необходимо сделать и повороты - туда, куда пуля двигается. Окей, сделаем 
``` cs
bullet.directionMoveTo = transform.right 
```
All in all, to make a system with possibility to make an another unit - moving not from left, but from right, you should:
1. use Mathf.Sign(transform.localScale.x);  - it changes localScale.x if it's - to +, so you can make directions calmly
2. the most important - CHANGE ROTATION BEFORE MOVEMENT. It helped a lot, i didn't think it would be so easy

It's a description of problem with bullets in flappy bird. [See on github](https://github.com/42BOBREEK/FlappyBird/blob/main/Assets/Scripts/Generic/Bullet.cs)
