new - [[Модификаторы доступа|модификатор доступа]], **не переопределяющий** метод родительского класса, а наоборот - **скрывающий** реализацию у дочернего объекта от [[Полиморфизм|полиморфизма]]. 

Например, **скрытие** метода Speak() у дочернего класса собаки от **полиморфизма**
``` cs
public class Animal
{
    public void Speak() => Console.WriteLine("Ааа");
}

public class Dog : Animal
{
    public new void Speak() => Console.WriteLine("Гав!");
}
```
**Переменная** типа **Animal**, но **объект Dog** выведет Aaa, т.к. мы **скрыли** от полиморфизма реализацию Speak() у собаки.
``` cs
Animal a = new Dog();
a.Speak(); // ❌ Выведет "Ааа"
```
Но если создать **переменную** уже с типом **Dog** и с объектом **Dog** в ней, полиморфизма **не будет** и собака скажет гав.
``` cs
Dog d = new Dog();
d.Speak(); // ✅ Выведет "Гав!"
```
